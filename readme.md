# Compte rendu du rapport du projet Onsie:

Le dossier SQL contient les scripts SQL pour créer et remplir la base de données utilisée par l'application.
Le dossier PHP est la racine du projet PHP permettant d'afficher les pages web.

Le fichier "PHP/config/db.ini" est initialement vierge et doit être remplis avec les informations de la base de données utilisée. Un message d'erreur sera affiché par l'application si le fichier manque ou si les informations ne permettant pas la connexion.

L'application comporte un menu permettant d'accéder et de tester les différentes fonctionnalités demandées.
